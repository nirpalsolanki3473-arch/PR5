# ✈️ Flight Delay Analysis Dashboard (Power BI)

## 📌 Project Overview

The **Flight Delay Analysis Dashboard** is an interactive Power BI project designed to analyze airline performance, flight delays, airport operations, and delay trends. The dashboard enables users to monitor key performance indicators (KPIs), compare airlines, identify delay causes, and explore flight patterns through interactive visualizations.

This project demonstrates data cleaning, Power Query transformations, DAX calculations, data modeling, forecasting, and dashboard design using Microsoft Power BI.

---

# 🎯 Project Objectives

- Analyze overall flight performance.
- Monitor departure and arrival delays.
- Compare airline operational efficiency.
- Identify high-delay airports and routes.
- Explore delay trends over time.
- Provide interactive filtering for better decision-making.
